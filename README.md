# Wayne's Professional Portfolio Website

A modern, responsive portfolio website showcasing professional skills and projects.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling, hover effects, and form validation
- **Performance Optimized**: Fast loading with minimal dependencies
- **SEO Friendly**: Semantic HTML structure

## Sections

1. **Hero Section**: Professional introduction with call-to-action buttons
2. **About**: Personal story and key statistics
3. **Projects**: Featured work with technology tags and links
4. **Skills**: Technical skills organized by category
5. **Contact**: Contact form and professional details

## Customization Guide

### 1. Personal Information
Update the following in `index.html`:
- Replace "Wayne" with your full name
- Update the professional title and description
- Add your contact information
- Replace placeholder project descriptions

### 2. Projects
For each project in the projects section:
- Replace project titles and descriptions
- Update technology tags
- Add live demo and GitHub links
- Replace project images (add to `images/` folder)

### 3. Skills
Update the skills section with your technologies:
- Frontend: HTML, CSS, JavaScript frameworks
- Backend: Server technologies and languages
- Database: Database systems you work with
- Tools: Development tools and platforms

### 4. Styling
Customize colors in `styles.css`:
- Primary color: `#2563eb` (blue)
- Accent color: `#fbbf24` (yellow)
- Background gradient: Update hero section gradient

### 5. Images
Create an `images/` folder and add:
- Profile photo (300x300px recommended)
- Project screenshots (16:9 aspect ratio recommended)
- Favicon (add to HTML head)

## Getting Started

1. Open `index.html` in your browser to view the website
2. Customize the content with your information
3. Add your project images to the `images/` folder
4. Update links to your actual projects and social media
5. Deploy to your preferred hosting platform
